## Prompt Structure

### Basic Formula
```
[Subject] + [Style] + [Lighting] + [Composition] + [Quality modifiers]
```

### Example Prompts

**App Icon**:
```
Minimal iOS app icon, [concept], gradient background,
rounded corners, modern flat design, 3D subtle depth,
high quality render, centered composition
```

**Hero Image**:
```
[Scene description], cinematic lighting, wide angle,
professional photography style, 8k resolution,
shallow depth of field, golden hour
```

**Illustration**:
```
[Subject] in the style of [artist/style],
vector illustration, clean lines,
limited color palette, white background
```

## Best Practices

1. **Be Specific**: "A golden retriever" vs "A dog"
2. **Layer Details**: Start simple, add complexity
3. **Use References**: "in the style of..."
4. **Iterate**: Refine prompts based on outputs
5. **Negative Prompts**: Exclude unwanted elements
